## 项目规范

- 存放内容：代码均存放在custom_addons文件夹

### 项目进度说明文件
- 内容要求：模块进度说明，按照时间顺序展开
- 命名规范：模块名_XX模块.md（eg：sale_extend_销售管理模块.md）
### 项目评估文件
- 评估内容：针对某模块实施+二次开发所需的时间评估
- 命名规范：wan_项目评估.md(eg:wan_项目评估.md)
### images文件夹
- 存放内容：图片文件
- 命名规范：.模块名_XX模块_images
